Static addSequence

  • addSequence(data: MarkovChainSequenceDTO, sequence: string[], insert?: MarkovChainInsertType, copy?: boolean): MarkovChainSequenceDTO
    • Defined in structures/markov.ts:186

    Adds or inserts a Sequence into a Markov Chain DTO.

    Parameters

    • data: MarkovChainSequenceDTO

      A Markov Chain data transfer object.

    • sequence: string[]

      The sequence to be added.

    • insert: MarkovChainInsertType = false

      Determines how sequences should be inserted. If false, delimiters will be prepended and appended to the sequences. "start" or setting true will only prepend the start delimiter, while "end" will append the end delimiter. "middle" will not add any delimiters.

    • copy: boolean = true

      Whether or not an immutable deep copy of the dto should be returned.

    Returns MarkovChainSequenceDTO
